# Manual de Usuario 

# BarberStudio

## Versión 1.0

### Junio 2026

---

# 1. Introducción

BarberStudio es un sistema web diseñado para facilitar la administración de citas, clientes, anticipos, barberos y reportes de una barbería.

Este manual tiene como objetivo orientar a los usuarios en el uso correcto de las funcionalidades principales del sistema.

---

# 2. Acceso al Sistema

### Iniciar Sesión

1. Abrir el navegador web.
2. Acceder a la dirección del sistema.
3. Ingresar usuario y contraseña.
4. Presionar el botón **Iniciar Sesión**.

Si las credenciales son correctas, el sistema mostrará el panel principal.

---

# 3. Menú Principal

El menú principal contiene las siguientes opciones:

* Dashboard
* Clientes
* Citas
* Anticipos
* Barberos
* Reportes
* Cerrar Sesión

---

# 4. Gestión de Clientes

### Registrar Cliente

1. Seleccionar la opción **Clientes**.
2. Presionar **Nuevo Cliente**.
3. Capturar los datos solicitados:

   * Nombre
   * Teléfono
   * Correo electrónico
4. Presionar **Guardar**.

### Consultar Cliente

1. Ingresar el nombre o teléfono en el buscador.
2. Seleccionar el cliente deseado.

El sistema mostrará su historial de citas y servicios.

---

# 5. Gestión de Citas

### Registrar Cita

1. Seleccionar **Citas**.
2. Presionar **Nueva Cita**.
3. Seleccionar:

   * Cliente
   * Fecha
   * Hora
   * Servicio
   * Barbero
4. Presionar **Guardar**.

### Modificar Cita

1. Buscar la cita en la agenda.
2. Seleccionar **Editar**.
3. Realizar los cambios necesarios.
4. Guardar cambios.

### Cancelar Cita

1. Seleccionar la cita.
2. Presionar **Cancelar**.
3. Confirmar la acción.

---

# 6. Gestión de Anticipos

### Registrar Anticipo

1. Abrir la cita correspondiente.
2. Seleccionar **Registrar Anticipo**.
3. Capturar:

   * Monto
   * Método de pago
4. Guardar información.

Las citas con anticipo registrado aparecerán identificadas visualmente dentro de la agenda.

---

# 7. Gestión de Barberos

### Registrar Barbero

1. Seleccionar **Barberos**.
2. Presionar **Nuevo Barbero**.
3. Capturar la información requerida.
4. Guardar registro.

### Consultar Disponibilidad

1. Seleccionar el nombre del barbero.
2. Revisar las citas asignadas en la fecha seleccionada.

---

# 8. Reportes

### Generar Reporte

1. Seleccionar **Reportes**.
2. Elegir el tipo de reporte:

   * Citas realizadas.
   * Citas canceladas.
   * Ingresos por anticipos.
3. Seleccionar rango de fechas.
4. Presionar **Generar**.

---

# 9. Cerrar Sesión

Para finalizar el uso del sistema:

1. Presionar **Cerrar Sesión**.
2. Confirmar la salida.

---

# 10. Recomendaciones

* Mantener la información de clientes actualizada.
* Registrar los anticipos inmediatamente después de recibirlos.
* Cerrar sesión al finalizar la jornada.
* Verificar diariamente la agenda de citas.

---

# 11. Soporte

En caso de presentar problemas con el sistema, contactar al administrador o responsable del proyecto para recibir asistencia.
