# 2. Descripción del Proyecto

BarberStudio es una plataforma web diseñada para administrar las operaciones principales de una barbería, permitiendo gestionar citas, clientes, anticipos, barberos y reportes operativos.

El sistema busca mejorar la organización de las actividades diarias, reducir errores administrativos y facilitar el seguimiento de clientes y servicios.

---

# 3. Alcance

El proyecto contempla las siguientes funcionalidades:

### Gestión de Citas

* Registro de citas.
* Modificación de citas.
* Cancelación de citas.
* Consulta de agenda.

### Gestión de Anticipos

* Registro de anticipos.
* Consulta de pagos realizados.
* Identificación visual de citas confirmadas.

### Gestión de Clientes

* Registro de clientes.
* Consulta de historial.
* Búsqueda rápida.

### Gestión de Barberos

* Registro de barberos.
* Consulta de disponibilidad.
* Asignación de citas.

### Reportes

* Reporte de citas realizadas.
* Reporte de citas canceladas.
* Reporte de ingresos por anticipos.

---

# 4. Fuera de Alcance

Las siguientes funcionalidades no forman parte de esta versión:

* Aplicación móvil nativa.
* Facturación electrónica.
* Pasarelas de pago en línea.
* Integración con redes sociales.
* Integración con sistemas externos.
